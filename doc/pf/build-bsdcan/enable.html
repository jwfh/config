How To Enable?

OpenBSD 4.6 onwards: on by default

Others: rc.conf PF="YES" (or rc script)

all: enable or check if enabled

$ sudo pfctl -ef /etc/pf.conf

(pfctl "Enable, load From /etc/pf.conf")

If already enabled:

pfctl: pf already enabled
