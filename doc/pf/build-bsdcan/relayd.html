relayd

Problem: machines go down, rdr doesn't compensate.

relayd detects host state (up/down) interacts with tables, adds/removes hosts

Configured via pf.conf(5) and relayd.conf(5)

Introduced in OpenBSD 4.1 as hoststated, renamed relayd in OpenBSD 4.3, actively developed
